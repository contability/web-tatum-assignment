# HOW TO RUN

---

## 의존성 패키지 설치

```bash
# use npm
npm i

# use yarn
yarn
```

## 프로젝트 실행

### 개발자 모드 실행

```bash
# use npm
npm run dev

# use yarn
yarn dev
```

### 프로젝트 빌드

```bash
# use npm
npm run build

# use yarn
yarn build
```

### 빌드된 프로젝트 실행

```bash
# use npm
npm run start

# use yarn
yarn start
```
